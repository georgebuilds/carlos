Carlos

The most { hard-working } coding agent. committed

Switch between personal, work, and side-project frames with Ctrl+F. Each frame has its own provider, rules, and plain-markdown memory you can grep from your shell or open in Obsidian. carlos stays inside the frame you're in unless you ask him to cross. One Go binary, no setup ritual.

Direct download

around 30 MB binary no Python, no Docker GPL-3.0

Install

Three ways in. Pick one. They all land the same binary under /usr/local/bin/carlos or the path you choose.

Homebrew (macOS / Linux)

$brew install georgebuilds/tap/carlos

The tap pushes a fresh formula on every stable release.

Tarball

macOS Apple Silicon · arm64 macOS Intel · x86_64 Linux arm64 Linux x86_64

Extract and drop carlos into your $PATH. All releases & checksums →

Verify

$carlos version

Should print a tagged version. If not, your $PATH isn't seeing the binary.

Getting started

Four steps. The TUI runs onboarding the first time you launch it.

  1. 1 · Run onboarding

    ~30 seconds. Your name, a provider (Anthropic, OpenAI, OpenRouter, Ollama, or Gemini), a model, the daemon yes/no, an optional Obsidian vault path. Everything lands in ~/.carlos/config.yaml at mode 0600. 

    $carlos

  2. 2 · Chat with carlos

    Type a question. carlos answers, calls tools when needed (prompting on the risky ones), and keeps the transcript in a SQLite event log at ~/.carlos/state.db. Ctrl+F opens the frame switcher; /help lists slash commands.

  3. 3 · Or skip the TUI

    carlos please "<prompt>" is a one-shot tool-use loop that stays in your scrollback (with a live 3-row status panel while it runs). carlos research "<question>" runs a six-phase pipeline and saves a cited report under ~/.carlos/research/. 

    $carlos please "summarize the last 5 commits"

  4. 4 · Background work

    carlos daemon enable installs a per-user launchd / systemd unit. carlos schedule add "every weekday at 9am ..." queues a recurring run. The daemon notifies on completion. 

    $carlos schedule add "every weekday at 9am summarize my inbox"

What you'll see

Five surfaces, faithfully reproduced from the real TUI. The research panel runs live in your browser.

Keys worth knowing

!<cmd>

/agents
open the sub-agent manage view
/whoami
current frame, mode, provider, model
/permissions
layered approval state & audit log
/compact
summarize the transcript, keep going
